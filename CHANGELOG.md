# Changelog

## 0.1.0

- Initial public release
- `.notes` language: tasks, time logging (`+`), tags (`=tag`)
- Month and day headers with automatic time summaries
- Indent structure: month → day → tasks
- Commands: toggle done, add 15 minutes, recalculate
- Tag colors, URL highlighting, hover summary per tag
- Status bar with remaining time for the current day
